# Drive Component TEST

Cool test for Drive process job interview.

## Run Locally

Clone the project

```bash
  git clone https://github.com/tonatiuh19/drive-component-project
```

Go to the project directory

```bash
  cd drive-component-project
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
